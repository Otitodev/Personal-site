# Product Overview

AstroZen is a minimalist, accessible, and responsive personal portfolio template built with Astro and TailwindCSS. The project focuses on clean design, performance, and developer experience.

## Key Features

- **Minimalist Design**: Clean, simple aesthetic with mobile-first responsive layout
- **Blog System**: Content collection-based blog with Markdown support and frontmatter metadata
- **Projects Showcase**: Portfolio section with featured projects, technology tags, and project status
- **Performance Focused**: Optimized for Lighthouse scores and fast loading
- **Accessibility First**: Semantic HTML, proper heading hierarchy, keyboard navigation
- **SEO Optimized**: Meta descriptions, social sharing, canonical URLs

## Target Use Case

Personal portfolio websites for developers, designers, and creative professionals who want a fast, accessible, and easy-to-maintain site with blog and project showcase capabilities.

## Content Strategy

- Blog posts in `src/content/blog/` with rich frontmatter (title, description, tags, dates, hero images)
- Projects in `src/content/projects/` with technology stacks, live URLs, GitHub links, and status tracking
- Featured content highlighting for important projects
- Draft system for content management
